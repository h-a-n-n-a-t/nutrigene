# 🧬 NutriGene - Genetic Mutation Diagnosis & Nutrition System

## Overview

NutriGene is a web-based application that identifies genetic mutations related to diseases like Type 1/2 Diabetes, Obesity, Lactose Intolerance, and Hyperlipidemia. It uses an interactive questionnaire to assess symptoms and provides personalized nutrition recommendations based on genetic profiles.

## Features

- ✅ **Multi-Gene Analysis**: Test for 16+ genetic mutations
- ✅ **Symptom-Based Diagnosis**: Interactive questionnaire system
- ✅ **Personalized Nutrition Plans**: Customized diet recommendations
- ✅ **Risk Assessment**: Confidence scoring and risk level classification
- ✅ **Comprehensive Reports**: Download diagnostic reports
- ✅ **Responsive Design**: Works on desktop and mobile devices

## Supported Genes

### Diabetes & Metabolism
- CTLA4, IL6, VEGFA, CDKN2A, PPARG, FTO, INS, TCF7L2

### Obesity & Appetite
- MC4R, LEP, LEPR

### Lactose Intolerance
- LCT, MCM6

### Cholesterol & Lipids
- APOE, CETP, PCSK9
